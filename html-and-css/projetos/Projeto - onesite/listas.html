Tipos de listas

Listas organizadas

Para usar as listas organizadas se usa a tag <ol> onde você pode definir o tipo(type - romano, números e letras), onde inicia(start) e para definir os pontos dentro da lista use a tag <li>.
Exemplo:

  1. - acordar
  2. - escovar os dentes
  3. - lavar roupas
  4. - ir pro trabalho
  5. - voltar pra casa
  6. - dormir

Listas desorganizadas

Parecido com a lista organizada mas se usa a tag <ul> e os tipos são "disc, circle, square".
Exemplo:

  • pão
  • leite
  • ovos
  • maçã

E é claro que você também pode misturar as duas listas entre si.


Listas de definição

Se usa a tag <dl> e dentro dela a tag <dt> vem primeiro e as tags <dd> vem depois.
Exemplo:

Html
Linguagem de marcação para criação do conteúdo do site
CSS
Linguagem de marcação para o design de um site
JavaScript
Linguagem de programação para a criação de interatividade de um site
