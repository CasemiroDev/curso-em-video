Códigos e citações

Códigos fonte

A tag <code> deixa o texto em forma de código.
Exemplo:

document.getElementById é escrito em linguagem JavaScript.

(obs: visualize o código para melhor entendimento).


Mas pode ser usado a tag <pre> antes do <code> para definir em formato de código em si.
Exemplo: 

        num = int(input('Digite um número'))
     if num % 2 == 0:
      print('ok, acept!')

Citações

Existem duas tags para citações

  • <q>
  • <blockquote>

A primeira deixa o texto com aspas, deste jeito: Falou e disse.
Já a segunda faz uma citação de forma discreta(Não esqueca de usar o "cite:" dentro do <blockquote>), deste jeito:

Falou e disse.

(obs: visualize o código para melhor entendimento)


Abreviações e texto invertido

Abreviações

Nas abreviações se usa a tag <abbr>(não esqueca de nomear com um título usando title dentro do <abbr>), deste jeito:

I love you.

Arraste o mouse por cima da abreviação.


Texto invertido

Para inverter o texto use a tag <bdo> com o "dir" dentro para determinar a direção.
Exemplo:

Nós te amamos.


Principais formatações e adicionais

Negrito

Para deixar em negrito apenas use a tag <b> deste jeito:
We love you.

(obs: visualize o código para melhor entedimento).

Itálico ou ênfase

Para deixar em itálico use <i>(com semântica) ou <em>(sem semântica).
Exemplo:

You are cool
Você é demais


Formatações adicionais

Use a tag <mark> para marcar algo, podendo até definir a cor.
Exemplo:

Play with me please


Para usar o texto deletado use a tag <del>.
Exemplo:

Ainda bem


Para usar o texto inserido use a a tag <ins>.
Exemplo:

Oh no


Para usar textos sup e sub escritos use as tags <sup> e <sub>.
Exemplos:

2020

H2O


Para usar textos grandes e pequenos use as tags <big> e <small>.
Exemplos:

Olá, Mundo
Hello, World!
