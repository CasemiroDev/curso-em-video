CSS

CSS Inline

O CSS inline é quando o código de estilos são feitos na própria tag.
Exemplo:

<p> style="font-family:... color:... and etc."

CSS Internos

É quando os códigos CSS são feitos pela cabeça usado a tag <style> e dentro dela todas as outras tags que estão no body que você for personalizar.
Exemplo:

<meta charset="utf-8" />
<title>Aprenda CSS</title>
<style>
body {
background-color:...
etc...

h1{
color:....
font-family:...
}
</style>

CSS Externo

É como o CSS interno mas a diferença é que é feito apenas o código em uma folha separada(deve ser criada assim nomedoarquivo.css), após isso você deve chamar a folha css em seu código html desse jeito:


<meta charset="utf-8" />
<title>Aprenda CSS</title>
<link rel="stylesheet" href="pasta/nomedoarquivo.css" />

Harmonia das cores

Harmonia - foco do estudo do círculo cromático | simetria - cores (o que deixa algo bonito) - cores primárias tem harmonia grosseira - cores secundárias tem harmonia grosseira - cores terciárias são todas as outras misturas de cores (amarelo-esverdeado ou amarelo-alaranjado) - 12 cores no círculo cromático | lado esquerdo - cores frias | lado direito - cores quentes - geralmente se usa cor pirmária da logo do site - cores complementares - cores que tem maior contraste com outras cores | cores análogas - que estão perto uma da outra no círculo cromático - cores análogas relacionadas - em duas cores você tem um degradê destas duas você pula uma cor e pega a tarceira cor depois da pulada, isso para qualquer direção - cores intercaladas - pega uma cor, pula uma pra alguma direção e escolhe a outra, pula mais uma, e pega a terceira cor depois da pulada - três cores intercaladas - triádicas (pula 3 cores e pega outra cor) - cores enquadradas (pula duas cores e pega outra cor) - cores tetrádricas (achar a cor complementar de duas cores formando 4 cores) - monocromia - única cor mudando apenas saturação e brilho.
